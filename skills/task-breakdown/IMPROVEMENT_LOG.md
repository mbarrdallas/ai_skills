# Task Breakdown Skill - Improvement Log

Track improvements made to this skill based on real-world usage.

## 2026-06-27: Add checkbox update requirement

**Issue:** Orchestrator was marking tasks as "✅ COMPLETE" in headers but not updating the individual acceptance criteria checkboxes `[ ]` → `[x]`.

**Learning:** Task plans with checkboxes need explicit guidance on when and how to update them.

**Improvement:** Added to SKILL.md:
- Section on "Tracking Progress" 
- Explicit instruction: "When a task completes, mark ALL acceptance criteria checkboxes as checked"
- Template now includes "Completed" and "Iterations" fields
